# Налаштування iPhone Shortcut для BlockMate

Цей документ описує, як створити Shortcut на iPhone для швидкого доступу до BlockMate бота.

## Варіант 1: Відкриття чату з ботом

### Крок 1: Отримайте username вашого бота

1. Відкрийте Telegram
2. Знайдіть вашого бота (створеного через @BotFather)
3. Відкрийте профіль бота
4. Запам'ятайте username (наприклад, `blockmate_bot`)

### Крок 2: Створіть Shortcut

1. Відкрийте додаток **Shortcuts** на iPhone
2. Натисніть **"+"** в правому верхньому куті
3. Натисніть **"Add Action"**
4. Пошукайте та додайте **"URL"**
5. Введіть URL у форматі:
   ```
   tg://resolve?domain=your_bot_username
   ```
   (замініть `your_bot_username` на username вашого бота)
6. Додайте дію **"Open URLs"** після URL
7. Натисніть **"Next"** в правому верхньому куті
8. Назвіть Shortcut (наприклад, "BlockMate")
9. Натисніть **"Done"**

### Крок 3: Додайте на головний екран (опціонально)

1. Відкрийте створений Shortcut
2. Натисніть на **"..."** в правому верхньому куті
3. Натисніть на іконку налаштувань внизу
4. Оберіть **"Add to Home Screen"**
5. Налаштуйте іконку та назву
6. Натисніть **"Add"**

## Варіант 2: Надсилання повідомлення з предзаповненим текстом

Цей варіант відкриває чат з ботом та автоматично надсилає повідомлення.

1. Створіть новий Shortcut
2. Додайте дію **"Text"** з текстом:
   ```
   Хочу перевірити соцмережу
   ```
3. Додайте дію **"Share"**
4. Оберіть опцію **"Telegram"** та вашого бота
5. Налаштуйте назву та іконку
6. Збережіть

## Варіант 3: Використання Siri

1. Створіть Shortcut (будь-який з варіантів вище)
2. Відкрийте налаштування Shortcut
3. Натисніть **"Add to Siri"**
4. Записуйте фразу-активацію (наприклад, "Перевірити соцмережу")
5. Тепер ви можете викликати бота через Siri, сказавши вашу фразу

## Варіант 4: Через Control Center

1. Відкрийте **Settings** → **Control Center**
2. Додайте **"Shortcuts"** до Control Center
3. Створіть Shortcut (варіант 1 або 2)
4. Тепер ви можете швидко викликати бота з Control Center

## Рекомендації

- **Назва**: Використовуйте коротку назву, наприклад "BlockMate" або "Помічник"
- **Іконка**: Оберіть яскраву іконку для швидкого розпізнавання
- **Колір**: Можна налаштувати колір іконки в налаштуваннях Shortcut

## Приклад використання

1. Ви хочете відкрити Instagram
2. Натискаєте на Shortcut на головному екрані
3. Відкривається Telegram з вашим ботом
4. Ви описуєте ваш запит: "Хочу відкрити Instagram на 10 хвилин"
5. Бот аналізує та дає рекомендацію
6. Ви приймаєте рішення на основі відповіді бота

## Вирішення проблем

### Shortcut не відкриває бота
- Перевірте, чи правильно введено username бота
- Переконайтесь, що бот запущений та працює
- Спробуйте перезапустити додаток Shortcuts

### Помилка при відкритті Telegram
- Переконайтесь, що Telegram встановлений на iPhone
- Спробуйте використати повний URL: `https://t.me/your_bot_username`

### Siri не розпізнає команду
- Переконайтесь, що ви додали Shortcut до Siri
- Використовуйте чітку та унікальну фразу
- Перевірте, що Siri увімкнена для додатку Shortcuts
